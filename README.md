# Spring Boot REST API

This project is a basic backend application built using Spring Boot.

## Features

* CRUD operations
* REST API development
* JSON request/response handling

## Tech Stack

* Java
* Spring Boot
* Maven

## API Endpoints

* GET /user
* POST /user
* PUT /user
* DELETE /user/{id}

## How to Run

1. Clone the repository
2. Open in IntelliJ
3. Run the application
4. Test using Postman
